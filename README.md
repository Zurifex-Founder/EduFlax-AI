# EduFlax AI

**Your Personal Student AI Helper Agent**

Built by **Flax**

A fun, powerful CLI-based AI agent designed to help students study smarter, not harder.

## ✨ Features

- **Smart Q&A** — Ask any academic question and get helpful guidance
- **Personalized Study Plans** — Custom weekly schedules based on your subjects, available hours, and goals
- **Interactive Quizzes** — Test yourself on Python Basics, Algebra, Biology with instant scoring and explanations
- **Concept Explainer** — Clear breakdowns with simple versions, details, analogies, and real-world examples (photosynthesis, derivatives, AI, supply & demand, history, etc.)
- **Math Solver** — Real symbolic mathematics powered by SymPy (solve equations, derivatives, integrals, simplify expressions)
- **Study Tips & Motivation** — Fresh tips + motivational boosts every session
- **EduNews Digest** — Relevant 2026 student/education news with "why it matters" insights

## 🚀 How to Run

```bash
python EduFlax_Student_AI_Agent.py
```

No extra dependencies required for core features. For the full math solver:
```bash
pip install sympy
```
